## 🎓 技術的な工夫

### Tailwind CSS v4 対応

当初 `@apply` ディレクティブで実装しようとしたが、Tailwind v4の仕様変更により以下の問題が発生:

```
Error: Cannot apply unknown utility class `gap-2`.
```

**解決策:**
- `@apply` を使わず、純粋なCSSプロパティで実装
- カスタムプロパティ（CSS Variables）でテーマ管理
- Tailwindのユーティリティクラスはマークアップ側で使用

**利点:**
- ビルドエラーの回避
- より明確なスタイル定義
- メンテナンス性の向上
- パフォーマンスの最適化
