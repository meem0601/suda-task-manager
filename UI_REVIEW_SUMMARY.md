## 🚨 緊急対応が必要な項目（優先度：最高）

### 1. タッチターゲットサイズ不足

**問題点**:
- フィルターボタン: 約40px ❌（基準44px未満）
- AI提案ボタン（詳細画面）: 約36px ❌
- サブタスクチェックボックス: タップ領域20px ❌

**リスク**:
- タップミス頻発
- ユーザー体験低下
- Apple/Google のUIガイドライン違反

**改善方法**:
```tsx
// すべてのボタンに min-h-[44px] 追加
<button className="px-4 py-3 min-h-[44px] ...">

// チェックボックスはlabel全体をタップ領域に
<label className="p-4 min-h-[52px] ...">
  <input type="checkbox" className="w-6 h-6" />
</label>
```

**実装工数**: 1時間

---

### 2. アニメーション・フィードバック不足

**問題点**:
- ボタン押下時のフィードバックが弱い
- モーダルが突然出現（transition無し）
- タスクカードタップの反応が薄い

**リスク**:
- 操作感が鈍く感じる
- ボタンが押されたか不明
- モダンアプリと比べて古臭い印象

**改善方法**:
```tsx
// ボタンにスケールアニメーション
<button className="transition-all duration-150 active:scale-95 ...">

// モーダルにスライドアップ
<div className="animate-slideUp ...">

// カードにタップフィードバック
<div className="transition-all active:scale-[0.98] ...">
```

**実装工数**: 2時間
